## 3. ניהול ביקורות כתובות

### תהליך ביקורת כתובה
```
לקוח מגיש ביקורת → מופיעה בלשונית "ממתינות" → אתה מאשר/דוחה → מוצגת באתר
```

### פעולות זמינות בלשונית "ממתינות"
- **✅ אשר** — הביקורת עוברת ל"ביקורות מאושרות" ומוצגת בעמוד ההמלצות ובעמוד הבית
- **❌ דחה** — הביקורת עוברת ל"ביקורות שנדחו" ואינה מוצגת
- **✏️ ערוך** — עריכת שם, עיר, סוג שירות, טקסט ותמונה לפני אישור

### עריכת ביקורת
ניתן לערוך ביקורת מאושרת בכל עת מלשונית "ביקורות מאושרות":
- שינוי טקסט, שם, עיר, שירות
- החלפת תמונה (העלה תמונה חדשה — הישנה נשארת ב-Storage)

### מה מוצג בעמוד הבית?
- 3 ביקורות **5 כוכבים** נבחרות **אקראית** בכל רענון עמוד
- רק ביקורות **מאושרות** שאינן **נדחות**

### מה מוצג בעמוד ההמלצות?
- 6 ביקורות כתובות אחרונות, עם כפתור "הצג עוד" (6 נוספות בכל לחיצה)

## 4. ניהול ביקורות וידאו

### הוספת ביקורת וידאו (לשונית "➕ הוסף וידאו")
**אפשרות א׳ — קישור YouTube/Vimeo:**
1. הדבק קישור לסרטון
2. מלא שם, עיר, שירות, ציטוט
3. לחץ "פרסם"

**אפשרות ב׳ — העלאת קובץ וידאו:**
1. לחץ על אזור הוידאו ובחר קובץ (MP4, MOV, WEBM)
2. הסרטון יועלה ל-Supabase Storage (bucket: `review-videos`)
3. מלא פרטים ולחץ "פרסם"

> **חובה:** יש להזין **קישור** *או* **קובץ** — לא שניהם ולא אחד בלי השני.

### עריכת ביקורת וידאו
בלשונית "ביקורות וידאו" ← לחץ **✏️** על הכרטיס הרצוי.  
ניתן לשנות: שם, עיר, שירות, ציטוט, תמונת פרופיל, וידאו.

### מה מוצג בעמוד ההמלצות?
- 3 ביקורות וידאו אחרונות, עם כפתור "הצג עוד"
- YouTube/Vimeo — מוצגים עם תמונת ממוזערת, לחיצה מפעילה inline
- קבצי וידאו — מוצגים ישירות עם player מובנה

## 5. ניהול מאמרים

### הוספת מאמר (לשונית "➕ הוסף מאמר")

| שדה | הסבר |
|-----|-------|
| **כותרת** | חובה. מוצגת בכרטיס ובראש המאמר |
| **תקציר** | מוצג בכרטיס ברשימה — שמור קצר (1-2 משפטים) |
| **תוכן מלא** | עורך עשיר — ראה טיפים למטה |
| **קטגוריה** | ריביות ועדכונים / טיפים למשכנתא / שוק הנדל"ן |
| **תמונה** | מוצגת בראש הכרטיס (יחס 16:9 מומלץ) |
| **וידאו** | קישור YouTube/Vimeo *או* קובץ וידאו |
| **מפורסם** | סמן כדי להציג באתר. בטל סימון לשמירה כטיוטה |

### עורך הטקסט העשיר
| כפתור | פעולה |
|--------|--------|
| **B** | הדגשה (Bold) |
| *I* | נטייה (Italic) |
| <u>U</u> | קו תחתון |
| צבע טקסט | בחר צבע ← סמן טקסט ← לחץ על picker |
| סימון | רקע צבעוני לטקסט מסומן |
| ✕ נקה | הסרת כל העיצוב מהסלקציה |

> **טיפ לעיצוב:** תחילה סמן את הטקסט, ואז בחר את הפעולה.

### עריכת מאמר קיים
ב"מאמרים 📝" ← לחץ **✏️ ערוך** על המאמר.  
- **👁 תצוגה מקדימה** — מציגה את התוכן כפי שיופיע לקורא
- **מחק סרטון** — מסיר את הוידאו הקיים

### קטגוריות
| ערך | תווית באתר |
|-----|------------|
| `rates` | ריביות ועדכונים |
| `tips` | טיפים למשכנתא |
| `market` | שוק הנדל"ן |

### כיצד הקורא רואה מאמר?
לחיצה על כרטיס מאמר פותחת **עמוד קריאה מלא** (overlay) עם תמונה, תוכן מעוצב, וידאו וכפתור חזרה.

> **המלצה:** טפל בכל משוב תוך 24 שעות — לחץ "סמן כטופל" לאחר שיחזרת ללקוח.

## 7. Supabase — ניהול מסד הנתונים

**כתובת:** [https://supabase.com](https://supabase.com) → Project: `dwlcascxmgspmtpqbfmr`

### טבלאות
| טבלה | תוכן |
|------|-------|
| `reviews` | ביקורות כתובות (name, city, rating, text, image_url, approved, rejected) |
| `video_reviews` | ביקורות וידאו (name, city, video_url, video_file_url, quote) |
| `feedback` | משובים שליליים (what_bad, improve, email, phone, handled) |
| `articles` | מאמרים (title, summary, body, category, image_url, video_url, published) |

### Storage Buckets
| Bucket | תוכן | Policies |
|--------|-------|----------|
| `review-photos` | תמונות ביקורות כתובות | INSERT לכולם, SELECT לכולם |
| `review-videos` | קבצי וידאו ביקורות | INSERT לכולם, SELECT לכולם |
| `article-images` | תמונות מאמרים | INSERT לכולם, SELECT לכולם |

### גיבוי מומלץ
1. Supabase Dashboard → **Table Editor**
2. בחר טבלה → לחץ **Export** → CSV
3. בצע גיבוי **אחת לחודש** לפחות

### ניקוי Storage
קבצים שהועלו ונמחקו מהטבלה **אינם נמחקים אוטומטית** מה-Storage.  
אחת לכמה חודשים: Supabase → Storage → עיין בקבצים ומחק ידנית קבצים ישנים.

## 8. הגדרות טכניות חשובות

### שינוי סיסמה
ב-`admin.html`, שורה עם:
```js
const ADMIN_PASSWORD = '343436';
```
שנה לסיסמה חדשה וחזקה.

### הגדרת URL פרודקשן (לקישור הביקורת)
ב-`admin.html`, שורה עם:
```js
const SITE_URL = '';
```
לאחר פריסה לאינטרנט, מלא:
```js
const SITE_URL = 'https://your-domain.co.il';
```

### מפתחות Supabase
מופיעים בראש כל קובץ JS. **אין לשתף** את ה-Service Role Key.  
המפתח הנוכחי הוא `publishable` (anon) — בטוח לצד לקוח.

## 9. פריסה לאינטרנט (Deployment)

האתר הוא **HTML סטטי** — ניתן לפרוס בחינם בכל אחת מהפלטפורמות:

### אפשרות א׳ — Netlify (מומלץ)
1. גרור את תיקיית הפרויקט לאתר [netlify.com/drop](https://netlify.com/drop)
2. קבל URL אוטומטי (לדוג׳ `yechiel-mortgages.netlify.app`)
3. ניתן לחבר דומיין מותאם אישית

### אפשרות ב׳ — GitHub Pages
1. פתח חשבון GitHub ← צור Repository חדש
2. העלה את כל הקבצים
3. Settings → Pages → Source: `main branch`

### לאחר הפריסה
1. עדכן `SITE_URL` ב-`admin.html` לדומיין האמיתי
2. וודא שכל הקישורים בין עמודים תקינים
3. בדוק שהביקורות נטענות מ-Supabase

## 10. שגיאות נפוצות ופתרונן

### "שגיאה בשליחה, נסה שנית" בטופס ביקורת
**סיבה:** חסרה Policy ב-Supabase Storage  
**פתרון:** Supabase → Storage → bucket `review-photos` → Policies → הוסף `INSERT` ל-`anon`

### העלאת וידאו נכשלת (שגיאה 400)
**סיבה:** חסרה Policy ב-bucket `review-videos`  
**פתרון:** אותו הליך — הוסף `INSERT` ל-`anon` ב-`review-videos`

### ביקורות לא מופיעות בעמוד הבית
**סיבה:** אין ביקורות מאושרות עם 5 כוכבים  
**פתרון:** ודא שביקורות אושרו בפאנל ושהדגל `approved=true` קיים בטבלה

### הפאנל מבקש סיסמה שוב לאחר רענון
זה **תקין** — הפאנל שומר את הכניסה רק ב-sessionStorage (עד סגירת הדפדפן).

### מאמר נשמר אך לא מופיע באתר
ודא שתיבת **"מפורסם"** מסומנת בטופס ההוספה/עריכה.
